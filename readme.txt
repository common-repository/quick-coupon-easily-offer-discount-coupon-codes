== Description ==

Quick Coupon allows you to quickly and easy offer code or discount codes on products to your visitors.

In less than 2 minutes, you can setup your site with Quick Coupon and start offering discount codes in no-time.  Configure your discount codes based upon Posts or Pages, and then simply insert the Shortcode into any Page or Post to display discount code box.

View a demo online at:
[Quick Coupon Demo Site](http://quickcoupon.iintense.com "Quick Coupon Demo Site")

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
1. Click on Quick Coupons under the Settings dropdown and configure
1. Create a New Post or Page and Click the Green Icon in The Toolbar to create a new discount code shortcode.
